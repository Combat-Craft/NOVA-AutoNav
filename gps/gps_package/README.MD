## ğŸ”§ Troubleshooting

- Use `ls /dev/tty*` to find your port
- Add user permissions: `sudo usermod -a -G dialout $USER && sudo chmod 666 /dev/ttyACM0`
- Check if port is in use: `sudo lsof /dev/ttyACM0`
- Kill interfering process: `sudo kill -9 <PID>`
- Reset ROS2 daemon:
```bash
ros2 daemon stop
ros2 daemon start
```
